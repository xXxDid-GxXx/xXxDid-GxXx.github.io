Name of project

UX Audit

Key details

Role: UI/UX Designer

Goal: Create template for providing UX services for crypto companies

Tools: Figma, GPT

Timeline: 3 weeks

Methodology

Nielsen's Heuristic Analysis: Systematic review using 10 usability principles.

Flow-Specific Analysis: Deep dives into critical user journeys to pinpoint friction.

Scoring System

Minor (1): Cosmetic flaws. Impact: Mild frustration.

Moderate (2): Partial task failure. Impact: Delays, support tickets.

Critical (3): Total task failure. Impact: Lost revenue, churn.

Analysis Scope

Flows: Onboarding, trading, portfolio management.

Sections: Dashboard, transaction history, NFT keys.

Focus: High-traffic pages driving 80% of activity.

Why This Works

Clarity: Scannable sections with examples.

Actionability: Every finding ties to a solution.

Visual Appeal: Infographics break up text.

Example Audit Findings

Critical: Missing Confirmation for Card Removal

Flow: Managing payment methods in crypto wallet

Problem: Users could accidentally remove a payment card without confirmation.

Fix: Add confirmation dialog with undo option.

Example Audit Findings

Medium: Confusing Exchange Rate Display

Flow: Converting crypto to fiat

Problem: Users didn't realize exchange rate was delayed.

Fix: Indicate real-time vs. delayed rates with refresh option.

Infographic Summary of Findings

Visual breakdown of issues by severity across all user flows.

Example: Critical 25% · Medium 40% · Minor 35%

Flow-specific infographics help prioritize improvements by impact.
